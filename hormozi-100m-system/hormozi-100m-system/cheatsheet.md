# Cheatsheet — $100M Business System Decision Guide

## Quick-Start: Which Offer Type Should I Use?

```
START: What's your goal?
│
├─→ Turn strangers into customers ──→ ATTRACTION OFFER
│   ├─ Need sustained effort from customer? ──→ Win Your Money Back
│   ├─ Want massive lead volume? ──→ Giveaway
│   ├─ Have a premium version? ──→ Decoy Offer
│   ├─ Customer can buy more of same? ──→ Buy X Get Y Free
│   └─ Can deliver quick yes/no result? ──→ Pay Less Now or More Later
│
├─→ Get existing customers to spend more ──→ UPSELL OFFER
│   ├─ Offer reveals a natural next problem? ──→ Classic Upsell
│   ├─ Multiple products to recommend? ──→ Menu Upsell
│   ├─ Have a 5-10x premium version? ──→ Anchor Upsell
│   └─ Want to re-engage past customers? ──→ Rollover Upsell
│
├─→ Customer said no ──→ DOWNSELL OFFER
│   ├─ Can't afford lump sum? ──→ Payment Plan Downsell
│   ├─ Product delivers quick results? ──→ Trial With Penalty
│   └─ Some features aren't essential? ──→ Feature Downsell
│
└─→ Keep customers paying month after month ──→ CONTINUITY OFFER
    ├─ Software/content access? ──→ SaaS/Subscription
    ├─ Physical products that run out? ──→ Consumable Reorder
    ├─ Ongoing expertise needed? ──→ Service Retainer
    └─ Group/facility access? ──→ Membership
```

## The Value Equation Quick Reference

```
Offer Score = (Dream × Success) / (Time × Effort)

COMPONENT          | DIRECTION | HOW TO IMPROVE
───────────────────┼───────────┼──────────────────────────────
Dream Outcome      | ↑ MAX     | Paint specific, emotional end state
Success Likelihood | ↑ MAX     | Add bold guarantees + social proof
Time Delay         | ↓ MIN     | Engineer a 24-48hr quick win
Effort & Sacrifice | ↓ MIN     | Make it done-for-you or done-with-you
```

**Target: Offer Score > 1,000,000**

| Score Range | Meaning |
|---|---|
| < 10,000 | Weak offer — major rework needed |
| 10,000 - 100,000 | Decent offer — optimize weakest component |
| 100,000 - 1,000,000 | Strong offer — fine-tune for Grand Slam |
| > 1,000,000 | Grand Slam Offer — scale aggressively |

## Core Four Lead Generation: Which Method First?

```
START: What's your situation?
│
├─→ Have existing contacts/followers? ──→ WARM OUTREACH (start here)
│   └─ Script: "Hey [name], I'm doing [X] for [audience].
│       Know anyone who'd be interested?"
│
├─→ Need leads but have no audience? ──→ COLD OUTREACH
│   └─ Rule of 100: 100 personalized messages/day minimum
│
├─→ Want to build trust at scale? ──→ CONTENT (Create)
│   └─ Every post is a 24/7 salesperson. Compound over time.
│
└─→ Proven offer, ready to scale fast? ──→ PAID ADS
    └─ Only after offer converts through other 3 methods
```

**Implementation Order**: Warm Outreach → Content → Cold Outreach → Paid Ads → Lead Getters

## Downsell Decision Tree

```
Customer says NO
│
├─→ Why?
│   ├─ "Too expensive" ──→ PAYMENT PLAN DOWNSELL
│   │   └─ "How much can you put down today?"
│   │   └─ Charge MORE total for payment privilege
│   │
│   ├─ "Not sure it works" ──→ TRIAL WITH PENALTY
│   │   └─ "Try it 14 days. Cancel anytime."
│   │   └─ Let results speak for themselves
│   │
│   └─ "Don't need all of that" ──→ FEATURE DOWNSELL
│       └─ Remove non-core features, reduce price
│       └─ Keep the core transformation intact
│
└─→ Still no?
    └─ Offer ROLLOVER CREDIT toward future purchase
    └─ "No problem. Can I follow up when [trigger]?"
```

## The One-Page Advertising Plan (from $100M Leads)

1. **Who**: Define your ideal customer (demographics, psychographics, pain points)
2. **What**: Your lead magnet or core offer (use Value Equation to make it irresistible)
3. **Where**: Choose your Core Four channel(s) based on audience type
4. **When**: Set campaign timeline with built-in urgency/scarcity
5. **How Much**: Budget, target cost-per-lead, break-even customer value
6. **CTA**: Clear action + reasons to act now (scarcity + urgency + any reason)
7. **Follow-Up**: Lead Getter plan for scaling what works
